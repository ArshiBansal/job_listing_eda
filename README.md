# 🚀 Job Listings Excel Dashboard

## 📌 Overview
Welcome to the **Job Listings Excel Dashboard**! This interactive dashboard is designed to help job seekers, recruiters, and analysts explore job trends, salary distributions, and location-based insights—all within Microsoft Excel. Whether you're hunting for your next opportunity or analyzing industry patterns, this tool will provide valuable insights!

## 🎯 Key Features
✅ **Interactive Charts & Graphs** – Gain insights into job trends, salaries, and top hiring locations.  
✅ **Dynamic Filters** – Easily refine data by job categories, companies, and regions.  
✅ **Data-Driven Insights** – Make informed career decisions with clear visual representations.  
✅ **Excel-Only Solution** – No Power Query or external tools required—just formulas, pivot tables, and slicers!  

## 📊 Dataset
The dataset (`process.xlsx`) includes job listings with fields such as:
- **Job Title** (e.g., Software Engineer, Data Analyst)
- **Company Name** (e.g., Amazon, Infosys, Deloitte)
- **Location** (e.g., Bangalore, Mumbai, Delhi)
- **Salary Range & Average Salary (LPA)**
- **Experience Required** (e.g., 2-5 years, 5-10 years)
- **Skills Required** (e.g., Python, SQL, Machine Learning)
- **Application Deadline & Number of Applicants**

## 🛠 How to Use
1️⃣ **Open** the `process.xlsx` file in Microsoft Excel.  
2️⃣ **Navigate** to the dashboard sheet to explore job trends and insights.  
3️⃣ **Use slicers & filters** to refine the data view based on your preferences.  
4️⃣ **Analyze** key metrics using built-in visualizations to make data-driven decisions.  

## ⚙️ System Requirements
- **Microsoft Excel 2016 or later** (for best compatibility)
- **Basic Excel skills** (pivot tables, slicers, formulas)
